# Shell 01 - Piscine

## Overview

Shell 01 builds on Shell 00 by introducing more advanced file and system manipulation commands. Students will practice searching, counting, and creating files, working with groups and permissions, and automating tasks with scripts.

## Objectives

* Practice searching and counting files in directories
* Learn to manage groups and permissions
* Automate tasks with simple shell scripts
* Handle files and directories efficiently
* Reinforce testing and debugging of commands
