# C06 - Piscine

## Overview

C06 focuses on command-line arguments in C. Students will learn to handle program parameters, print them, reverse their order, and sort them alphabetically.

---

## Exercises

| Exercise | Description |
|----------|-------------|
| 00 | Exercise 00: ft_print_program_name |
| 01 | Exercise 01: ft_print_params |
| 02 | Exercise 02: ft_rev_params |
| 03 | Exercise 03: ft_sort_params |

---

## Objectives

* Access and use command-line arguments (`argc` and `argv`)
* Print program name and parameters
* Reverse the order of command-line arguments
* Sort parameters alphabetically
* Reinforce string manipulation and array handling
* Follow 42 Norm coding standards

---

## Testing and Validation

* Compile and run each program with different arguments
* Check output for correctness and formatting
* Test edge cases such as no arguments or special characters
