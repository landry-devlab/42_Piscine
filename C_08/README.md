# C08 - Piscine

## Overview

C08 introduces header files, macros, structures, and modular programming in C. Students will learn how to define custom types, use macros, and organize code across multiple files using headers.

| 05 | Exercise 05: ft_show_tab |

---

## Objectives

* Understand the role and structure of header files (`.h`)
* Use macros and constants effectively
* Define and manipulate structures (`struct`)
* Organize code using modular design
* Convert arrays of strings into structured data
* Display structured data correctly
* Follow 42 Norm coding standards

---

## Testing and Validation

* Compile all files together to ensure proper inclusion of headers
* Verify correct structure definitions and macro usage
* Test memory allocation and output formatting
* Ensure no memory leaks or compilation warnings
