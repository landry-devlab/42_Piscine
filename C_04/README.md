# C04 - Piscine

## Overview

C04 introduces basic input/output and number handling in C. Students will practice string length calculation, printing strings and numbers, converting strings to integers, and printing numbers in arbitrary bases.

---

| Exercise | Description |
|----------|-------------|

| 01 | Exercise 01: ft_putstr |
| 02 | Exercise 02: ft_putnbr |
| 03 | Exercise 03: ft_atoi |
| 04 | Exercise 04: ft_putnbr_base |

---

## Objectives

* Calculate the length of strings using `ft_strlen`
* Print strings and numbers to the standard output
* Convert strings to integers with `ft_atoi`
* Print numbers in arbitrary bases using `ft_putnbr_base`
* Reinforce proper handling of numeric and string data
* Follow 42 Norm coding standards

---

## Testing and Validation

* Compile and run each function to verify correctness
* Test edge cases such as empty strings, zero, negative numbers, and invalid bases
* Ensure output formatting matches expected results
