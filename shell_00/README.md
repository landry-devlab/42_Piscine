# Shell 00 - Piscine

## Overview

Shell 00 is designed to introduce basic shell commands and scripting concepts. Students will practice file navigation, management, text manipulation, and version control basics. Each exercise focuses on a specific skill, gradually increasing in complexity.

## Objectives

* Learn and practice essential shell commands (`ls`, `cd`, `mkdir`, `rm`, etc.)
* Understand file permissions and navigation
* Manipulate text files with `cat`, `grep`, `wc`, etc.
* Work with pipelines and redirections
* Introduce basic Git operations (commit, gitignore, diff)
* Practice writing simple shell scripts
* Reinforce debugging and testing of commands
