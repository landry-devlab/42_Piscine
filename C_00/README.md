# C00 - Piscine

## Overview

C00 is designed to introduce students to basic C programming concepts, including writing functions, using loops and conditionals, and manipulating simple output. Each exercise builds progressively on the previous one.

---

## Objectives

* Learn to write and compile simple C functions
* Practice using loops (`for`, `while`) and conditionals (`if`, `else`)
* Understand character and number output with `write`
* Reinforce incremental development and testing of functions
* Prepare for more advanced exercises in C01 and beyond
