# C01 - Piscine

## Overview

C01 builds on C00 by introducing pointers, arrays, and basic function manipulations. Students will practice passing variables by reference, manipulating arrays, and performing string operations. Each exercise focuses on mastering fundamental C programming techniques.

## Objectives

* Understand and practice pointers and variable referencing
* Manipulate arrays of integers
* Work with string operations and string length calculations
* Implement basic algorithms for reversing and sorting arrays
* Reinforce coding discipline according to the 42 Norm
