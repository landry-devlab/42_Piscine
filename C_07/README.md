# C07 - Piscine

## Overview

C07 focuses on dynamic memory allocation in C. Students will practice duplicating strings, creating arrays with `malloc`, and managing memory safely.

Be careful, ex03 is missing to reach the required score.

## Objectives

* Understand and use dynamic memory allocation with `malloc` and `free`
* Duplicate strings safely with `ft_strdup`
* Create integer arrays with `ft_range` and `ft_ultimate_range`
* Handle memory correctly to avoid leaks
* Reinforce pointer manipulation and memory management
* Follow 42 Norm coding standards

---

## Testing and Validation

* Compile and run each function to check correctness
* Test various sizes and ranges for arrays
* Verify proper allocation and deallocation of memory
