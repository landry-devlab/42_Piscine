# C05 - Piscine

## Overview

C05 focuses on mathematical functions and recursion in C. Students will implement factorials, powers, Fibonacci sequence, square roots, and prime number calculations, both iteratively and recursively.

## Objectives

* Implement mathematical functions iteratively and recursively
* Calculate factorials and powers
* Generate Fibonacci numbers
* Compute integer square roots
* Determine prime numbers and find the next prime
* Reinforce proper use of loops, conditionals, and recursion
* Follow 42 Norm coding standards

---

## Testing and Validation

* Compile and run each function to verify correctness
* Test edge cases such as zero, one, negative numbers, and large values
* Ensure output matches expected results
